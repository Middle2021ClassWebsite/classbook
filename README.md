##### HTML引用
关于HTML引用语法：`https://gitee.com/lycolovcoxin/classbook/tree/PictureHub/<文件类型>/<文件编号>?raw=true/`

文件类型：
`图片`：Picture
`视频`：Video

文件编号：
目标文件的编号（文件名）

> 注：用gitee的链接是为了方便HTML进行引用访问

文件编号可以通过CBtool进行查找，关于[下载链接](https://github.com/lovcoxin/ClassBook/releases/tag/v1.2.2)
